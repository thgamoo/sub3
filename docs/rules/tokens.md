# 게임 토큰

## 능력 카드

> 각 10장씩, 매력 카드는 40장 총 100장

- 검
- 매력 (부채)
- 마법 (지팡이)
- 선혈 (핏방울)
- 신성 (십자가)
- 독약
- 어둠

## 물건 카드

> 하나의 물건 더미로 섞어서 사용한다.
> 캐릭터 과거의 단서 27장 + 세계 진실 물건 10장 + 엘릭서 3장 = 총 40장.

### 세계 진실 물건

> 세계 "판 로만티카" 자체의 진실에 다가가는 물건. 총 5종 각 2장, 10장.
> 승천엔딩 조건에 사용된다 (서로 다른 4종 이상 필요. 같은 종류의 중복은 1종으로 카운트).

### 엘릭서

> 총 3장.

- 사용 시 `빈사` 상태에서 벗어난다. ([actions.md](actions.md)의 "모략" 참고)

## 금화 카드

> 총 25장. 거래 시에 사용할 수 있다.

- 금화 카드 5장당 타 플레이어와 거래 시 물건 카드를 하나 가져올 수 있다.
- 상세한 거래 규칙은 [actions.md](actions.md)의 "거래" 참고.

> 플레이버: 거래 조건이 만족스럽지 못하면 — 플레이어들은 악녀이기 때문에 — 뒷세계를 매수해서 물건을 옮기는 과정에서 파손시켜 버리는 것이다.
