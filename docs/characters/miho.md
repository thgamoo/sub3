# 마력없는 마법사 - 미호 (Miho)

## 기본 정보

- 구분: 서브 빌런 (플레이어 캐릭터)
- 시작지점: 중부
- 시작자원: 금화 1, 매력 2, 마법 2
- 모티프: 구미호

## 외모

- 검은 생머리
- 안경
- 검은 눈
- 요력 오오라로 구미호인 것을 연하게 표시

## 원래 운명

중앙 마탑에서 차별받던 그녀는 `북부의 마녀 - 이제벨`의 유혹에 넘어가 금단의 마법에 대한 지식을 익히게 되고, 마탑은 그녀에게 척살령을 내린다. 부상을 입고 도망치던 미호는 주인공 일행과 마주하게 되고, 결국 토벌당한다.

## 가능한 각성 능력

- 일반 각성 능력 전부

## 숨겨진 과거

사실 미호는 반인반요 여우요괴였으며, 체질적으로 마력 대신 요력을 운용할 수 있다.

## 과거의 단서

- "요마변환기"
- "호요족의 기록"
- "용사전기"
- "백요잡서"
