# 게임 종료 조건 (Ending)

## 1. 노멀엔딩 - 나는 살아있다.

- 1명을 제외하고 모든 플레이어 사망
- 능력을 각성한 상태.
- 모든 빌런이 처치된 상태.
- 충족 시 즉시 `멸망` 시나리오 중단.

## 2. 해피엔딩 - 저 오늘 결혼해요~

- 1명이 다음의 조건을 충족
  - 생존
  - 능력을 각성한 상태
  - 4명의 남주 중 한명과 결혼
  - 해당 남주의 통치권의 빌런 제거

## 3. 군림엔딩 - 짐의 발 앞에 조아리거라.

- 1명이 다음의 조건을 충족
  - 생존
  - 능력을 각성한 상태
  - 본인의 과거 진실 단서들을 3개이상 수집
  - 극동의 땅에서 황제의 증표 획득
  - 본 대륙으로 돌아와 황제선언 후 다음라운드 되기 (1라운드 필요)

## 4. 승천엔딩 - 본녀는 세계의 진실을 깨달았도다

- 1명이 다음의 조건을 충족
  - 본인 캐릭터에 맞는 능력을 각성한 상태
  - 서로 다른 세계 진실 물건 카드 4종 이상 수집 (같은 종류의 중복 카드는 1종으로 카운트)
  - 아카데미에서 승천의 문 소환
  - 승천

## 5. 멸망엔딩 - 내가 가질 수 없다면 부숴버릴거야.

- 모든 플레이어가 사망
- 빌런 포인트를 가장 많이 획득한 사람이 승리
